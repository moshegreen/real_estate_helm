# Client Applications

`apps/windows` is the React/TypeScript shell intended for a Windows desktop package through Tauri, Electron, or WebView2. It targets the local HTTP API exposed by `real-estate-helm-server`.

`apps/android` is the React Native companion scaffold for alerts, quick summaries, watchlist review, and approval workflows. It intentionally avoids full underwriting workflows, matching `plan.md`.

These scaffolds are committed as source contracts. Dependency installation and platform packaging are still pending production work.
